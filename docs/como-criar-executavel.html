🖥️ Electron - Desktop Nativo

💡 Recomendado para: Aplicações desktop profissionais com recursos nativos completos.
O que é o Electron?

O Electron permite criar aplicações desktop nativas usando tecnologias web (HTML, CSS, JavaScript). Usado por apps como VS Code, Discord, WhatsApp Desktop.

✅ Vantagens

  • Interface nativa do sistema
  • Acesso completo ao sistema de arquivos
  • Menus nativos e atalhos
  • Notificações do sistema
  • Auto-updater integrado
  • Cross-platform (Windows, Mac, Linux)

❌ Desvantagens

  • Arquivo maior (~100-200MB)
  • Maior consumo de RAM
  • Requer Node.js para desenvolvimento
  • Curva de aprendizado inicial

🚀 Passo a Passo

  1. Baixar o Projeto

    Faça download de todos os arquivos do projeto ou clone o repositório.

    git clone https://github.com/seu-usuario/sistema-bombas.git
cd sistema-bombas
  2. Instalar Dependências

    Execute o comando para instalar todas as dependências necessárias.

    npm install
    💡 Dica: Este comando pode demorar alguns minutos na primeira vez.
  3. Testar em Desenvolvimento

    Antes de gerar o executável, teste se tudo está funcionando.

    npm run electron-dev
  4. Gerar Executável

    Crie o executável para distribuição.

    npm run dist
    ✅ Resultado: O executável será criado na pasta dist-electron/

📱 PWA - Progressive Web App

💡 Recomendado para: Distribuição web com capacidade de instalação nativa.
O que é PWA?

Progressive Web Apps são aplicações web que podem ser instaladas como apps nativos através do navegador, funcionando offline e com recursos avançados.

✅ Vantagens

  • Arquivo muito menor
  • Atualizações automáticas
  • Funciona offline
  • Instalação via navegador
  • Cross-platform automático
  • Sem necessidade de app store

❌ Limitações

  • Recursos limitados do sistema
  • Dependente do navegador
  • Menos integração nativa
  • Suporte variável entre navegadores

🚀 Como Usar o PWA

  1. Acessar o Site

    Abra o sistema no navegador (Chrome, Edge, Firefox, Safari).

  2. Instalar como App

    Procure pelo botão "Instalar" na barra de endereços ou menu do navegador.

    💡 Localização do botão:
    • Chrome: Ícone de instalação na barra de endereços
    • Edge: Menu ⋯ → Apps → Instalar este site como um app
    • Firefox: Menu → Instalar
    • Safari: Compartilhar → Adicionar à Tela Inicial
  3. Usar como App Nativo

    Após instalação, o app aparecerá no menu iniciar/aplicações e pode ser usado offline.

⚡ Tauri - Alternativa Leve

💡 Recomendado para: Máxima performance e menor consumo de recursos.
O que é o Tauri?

Tauri é um framework moderno que usa Rust para criar aplicações desktop leves, com frontend web e backend nativo de alta performance.

✅ Vantagens

  • Executável muito pequeno (~10-20MB)
  • Baixo consumo de memória
  • Performance superior
  • Segurança avançada
  • Cross-platform nativo
  • Atualizações automáticas

❌ Desvantagens

  • Requer Rust instalado
  • Curva de aprendizado maior
  • Comunidade menor
  • Menos recursos prontos
  • Compilação mais lenta

🚀 Configuração Tauri

  1. Instalar Rust 
    # Windows/macOS/Linux
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Reiniciar terminal após instalação
source ~/.cargo/env
  2. Instalar Tauri CLI 
    npm install -g @tauri-apps/cli
  3. Inicializar Tauri no Projeto 
    cd seu-projeto
npm install --save-dev @tauri-apps/cli
npx tauri init
  4. Configurar tauri.conf.json 
    {
  "build": {
    "beforeBuildCommand": "npm run build",
    "beforeDevCommand": "npm run dev",
    "devPath": "http://localhost:5173",
    "distDir": "../dist"
  },
  "package": {
    "productName": "Sistema de Seleção de Bombas",
    "version": "1.0.0"
  },
  "tauri": {
    "allowlist": {
      "all": false
    },
    "bundle": {
      "active": true,
      "targets": "all",
      "identifier": "com.bombas.sistema",
      "icon": [
        "icons/32x32.png",
        "icons/128x128.png",
        "icons/icon.icns",
        "icons/icon.ico"
      ]
    }
  }
}
  5. Build do Executável 
    # Desenvolvimento
npx tauri dev

# Produção
npx tauri build
⚠️ Nota: Tauri requer mais configuração inicial, mas resulta em aplicações muito mais leves e rápidas. Ideal para projetos que priorizam performance.

🔧 Solução de Problemas

🚨 Problemas Comuns - Electron

❌ Erro: "npm install" falha

Soluções:

# Limpar cache do npm
npm cache clean --force

# Deletar node_modules e reinstalar
rm -rf node_modules package-lock.json
npm install

# Usar yarn como alternativa
npm install -g yarn
yarn install
❌ Erro de Permissão (Windows)

Solução:

# Executar no PowerShell como Administrador
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
❌ Erro de Certificado (macOS)

Solução:

# Desabilitar verificação de certificado para desenvolvimento
export CSC_IDENTITY_AUTO_DISCOVERY=false
npm run dist
❌ Executável não abre

Verificações:

  • ✅ Antivírus não está bloqueando
  • ✅ Todas as dependências foram instaladas
  • ✅ Build foi executado com sucesso
  • ✅ Ícones estão no local correto
# Testar em modo desenvolvimento primeiro
npm run electron-dev

# Verificar logs de erro
npm run dist -- --publish=never --debug

🚨 Problemas Comuns - PWA

❌ Botão "Instalar" não aparece

Verificações:

  • ✅ Site está sendo servido via HTTPS
  • ✅ Manifest.json está acessível
  • ✅ Service Worker está registrado
  • ✅ Ícones estão no tamanho correto
💡 Dica: Abra as DevTools (F12) → Application → Manifest para verificar se está tudo correto.
❌ PWA não funciona offline

Solução:

# Verificar se Service Worker está ativo
// No console do navegador
navigator.serviceWorker.getRegistrations().then(function(registrations) {
    console.log(registrations);
});

🚨 Problemas Comuns - Tauri

❌ Rust não está instalado

Solução:

# Instalar Rust
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Verificar instalação
rustc --version
cargo --version
❌ Build do Tauri falha

Verificações:

  • ✅ Rust está atualizado: rustup update
  • ✅ Dependências do sistema instaladas
  • ✅ tauri.conf.json está correto
# Instalar dependências do sistema (Ubuntu/Debian)
sudo apt update
sudo apt install libwebkit2gtk-4.0-dev build-essential curl wget libssl-dev libgtk-3-dev libayatana-appindicator3-dev librsvg2-dev

# macOS
xcode-select --install

# Windows - instalar Visual Studio Build Tools
